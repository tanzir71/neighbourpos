Workflow

Sell → learn → target. Orders flow into customer recency/frequency/spend so you can segment and act.

POS

Search inventory, build cart, attach a phone (or walk-in), and update status from new → preparing → ready/out → completed.

CRM

Customers are keyed by phone. View LTV estimate, order history, tags, and marketing opt-in status.

Campaigns

Build segments (filters), preview recipients, simulate impact, then export/queue messages. Opt-in is enforced by default and overrides are audited.
